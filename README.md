# frogBumper
 random video player made in unity urp 3d.
 All .mp4 files in a folder and every .mp4 in subfolders will be played shuffled. If all videos have played a new shuffled list is created and plays

## Keyboard Binds
 [a] [d] Playback Speed
 [w] [s] Volume
 [m] Mute
 [return] Menu Screen
 [ESC] Exit app
